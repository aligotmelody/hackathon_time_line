🚀 On-Demand Delivery App Prototype Timeline (MERN Stack)

✅ Core Features to Target

  • User Roles: Customer, Driver, Admin
  • Customer: Signup/login, place delivery order, track status
  • Driver: Signup/login, accept orders, update delivery status
  • Admin: View/manage all orders
  • Extras: Google Maps integration (basic), notifications (optional)
